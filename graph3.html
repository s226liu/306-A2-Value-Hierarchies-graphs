Value Hierarchy 3

Stakeholder 2: Medical Professionals (Indirect Stakeholder)
Intrinsic Value: Accuracy
  • Level 1 — Value
    Accuracy
    • Level 2 — Objective
      Present clinically reliable data.
      • Requirement
        Export reports must separate raw logged data from algorithmic predictions.
      • Requirement
        Each exported entry must include timestamp and cycle day number.
      • Requirement
        Export reports must include cycle length average and standard deviation.
      • Requirement
        Export reports must display number of cycles used for prediction.
    • Level 2 — Objective
      Prevent overconfident predictions.
      • Requirement
        Disable ovulation prediction when fewer than 3 complete cycles are logged.
      • Requirement
        Display confidence rating on all predicted outputs.
      • Requirement
        Automatically hide ovulation estimates when cycle variability exceeds threshold.
      • Requirement
        Replace definitive phrasing with probabilistic phrasing (e.g., “likely between”).
    • Level 2 — Objective
      Preserve data integrity in exports.
      • Requirement
        Mark edited entries with “edited on” timestamp in exported logs.
      • Requirement
        Include missing log days indicator in reports.
      • Requirement
        Export files must use standardized PDF format with structured headings.
      • Requirement
        Data exports must include timezone information.
    • Level 2 — Objective
      Reduce misinformation risk.
      • Requirement
        Prohibit diagnostic statements in automated insights.
      • Requirement
        Include visible disclaimer: “Not a diagnostic tool.”
      • Requirement
        Attach source references to health education content.
      • Requirement
        Add guidance encouraging users to consult clinicians for severe symptoms.
